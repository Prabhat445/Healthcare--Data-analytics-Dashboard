# Healthcare--Data-analytics-Dashboard
An interactive power BI dashboard analyzing patient demographics, disease
 Healthcare Data Analytics Dashboard 🏥📊

## Project Overview
This project focuses on analyzing healthcare and patient profile data to find key business insights using **Excel** and **Power BI**. The dashboard helps hospital administrators track patient counts, analyze disease trends, and monitor billing amounts effectively.

## Tech Stack Used
* **Data Cleaning & Transformation:** Power Query (Excel/Power BI)
* **Data Modeling:** Power BI Desktop
* **DAX Formulas:** Written custom DAX measures for Average Bill and Patient Counts.
* **Data Visualization:** Interactive charts, Pie charts, and KPI Cards.

## Key Insights from the Dashboard
* **Patient Profiling:** Tracked patient demographics based on Age and Disease categories.
* **Disease Breakdown:** Identified the top diseases (Heart, Viral, Dengue) with highest patient volumes using interactive Pie Charts.
* **Financial Analysis:** Modeled billing amounts (`Bill_Amount` and `Avg_bill`) across different rooms and room types (`Room_Type`).

## How to Use this Repository
1. Download the `Healthcare_Project.pbix` file.
2. Open it in Power BI Desktop to view the live data model and interactive features.
